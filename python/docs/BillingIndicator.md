# BillingIndicator

ITBIS tax billing indicator for a line item: - `0`: Non-billable (No facturable) - `1`: 18% ITBIS rate - `2`: 16% ITBIS rate - `3`: 0% ITBIS rate - `4`: Exempt (Exento) 

## Enum

* `ENUM_0` (value: `'0'`)

* `ENUM_1` (value: `'1'`)

* `ENUM_2` (value: `'2'`)

* `ENUM_3` (value: `'3'`)

* `ENUM_4` (value: `'4'`)
